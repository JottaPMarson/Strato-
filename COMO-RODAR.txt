🚀 STRATOPJ - COMO RODAR LOCALMENTE
=======================================

📋 INÍCIO RÁPIDO:

1. Frontend (porta 3000):
   npm run dev

2. Backend (porta 4000):
   # Sem pnpm. Use npm na pasta backend
   npm --prefix backend install
   npm --prefix backend run dev

📍 URLs:
• Frontend: http://localhost:3000
• Backend: http://localhost:4000/api

🔐 Login:
• Email: admin@stratopj.com
• Senha: 123456

=======================================

📦 PRÉ-REQUISITOS:
• Node.js 18+ (https://nodejs.org)
• npm ou pnpm

=======================================

🔧 INSTALAÇÃO:

1. Instalar dependências do frontend:
   npm install --legacy-peer-deps

2. Instalar dependências do backend:
   npm --prefix backend install

3. Iniciar frontend:
   npm run dev

4. Iniciar backend (em outro terminal):
   npm --prefix backend run dev

=======================================

🛠️ COMANDOS DISPONÍVEIS:

Frontend:
  npm run dev          # Iniciar frontend (porta 3000)
  npm run build        # Build para produção
  npm run start        # Iniciar produção

Backend:
  npm --prefix backend run dev      # Iniciar backend (porta 4000)
  npm --prefix backend run build    # Build backend
  npm --prefix backend run start    # Iniciar backend produção

=======================================

❗ PROBLEMAS COMUNS:

Porta 3000 ocupada:
  npm run dev -- --port 3001

Porta 4000 ocupada (backend):
  # Matar processo na porta 4000
  netstat -tlnp | grep :4000
  kill -9 <PID>

Erro de dependências frontend:
  rm -rf node_modules package-lock.json
  npm install --legacy-peer-deps

Erro de dependências backend:
  rm -rf backend/node_modules backend/package-lock.json
  cd backend && npm install

Backend não inicia:
  # Verificar se as dependências estão instaladas
  cd backend && npm install
  npm run dev:backend

Frontend não conecta com backend:
  # Verificar se backend está rodando
  curl http://localhost:4000/api/health

Node.js não encontrado:
  Baixar em: https://nodejs.org
